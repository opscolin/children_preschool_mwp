# KidLeap - 幼小衔接智能互动学习助手 🎒

欢迎来到 **KidLeap**！这是一款专为 3-7 岁儿童设计的**幼小衔接浸润式互动学习平台**。在设计上，我们秉持“保护孩子专注力、快乐启发”的教育理念，不含任何广告或繁杂的冗余干扰，为孩子们搭建起探究知识的亲切桥梁。

无论是拼音初学、口算熟练，还是国学启蒙，KidLeap 都通过精美的卡片设计、暖心和缓的童音评价，让孩子们在欢声笑语中完成每日的学习小目标。

---

## ✨ 核心特色模块

项目核心由三大幼小衔接刚需模块以及完善的家长后台构成：

### 1. 拼音乐园 🎵
* **分类学习卡片**：科学划分 **声母 (Initials)**、**韵母 (Finals)** 以及 **整体认读音节 (Syllables)** 三大体系，配备专属的声调示例词语（如 “ā - 阿姨”）。
* **智选童声发音**：摒弃传统生硬的机器播音。系统会自动探测并选用当前运行浏览器内最拟人化、具有自然少儿情感的音色（如微软 `Xiaoyi (晓伊)`、`Xiaoxiao (晓晓)` 或 Apple 优质中文女声）。
* **听音练习游乐场**：通过“看拼音，听读音，选拼音”的趣味选择题挑战，即时评估拼音的拼读、认读熟练度。

### 2. 口算大冒险 ➕
* **家长自主控制**：支持自定义数字上限（如 10 以内、20 以内等加减法）以及单次冒险关卡的题量，分阶段夯实基础。
* **安全大字键盘**：专门针对幼龄儿童指尖触屏、键位感知发育特点设计的超大黄金间距虚拟数字键盘，并支持退格修改与快速提交。
* **暖心语音激励**：答题结束时，会根据最终得分发出极具启发、和缓温柔的少儿专属赞美语音。

### 3. 古诗背诵阁 📖
* **精选传世经典**：内置《咏鹅》、《静夜思》、《春晓》等家喻户晓的唐诗精选。
* **每日星级任务**：每日智选 2-3 首不重复的诗词，通过自主“打分星级”（1至3颗星）标识孩子的背诵熟练度。
* **全新达标设置**：家长可以科学制定达标总星数目标，包含 **10星**、**20星**、**30星**、**50星** 四大档位，满足不同认读起跑线上孩子的阶梯式成长。

## 🎨 匠心工艺与体验优化

为了将用户体验推向极臻精美，我们在细节配置上下足了功夫：

* **🚫 全局无感滚动条**：对拼音列表、古诗对照表、测验面板等任何拥有长列表的容器，全部进行了底层 CSS 精细优化。**隐藏了所有突兀的灰色滚动条，但完美保留鼠标滑轮、触屏手指无阻顺滑滑动**，令整个界面宛如一张连续、澄净的书卷。
* **🧒 智能少儿音色架构**：
  * 深度调用现代浏览器（Edge/Chrome/Safari）底层 TTS 音频合成引擎。
  * 自动检索并适配高保真情绪类童音，若无则自适应调整默认配音的 **播放基础音调 (Pitch to 1.15x-1.25x)** 与 **语速比率 (Rate down to 0.72x-0.85x)**，使其更偏向童趣、温和、易辨识的教学主播语气，保护孩子稚嫩的听力和情绪。
* **📱 跨平台自适应容器**：提供高清手机、平板电脑以及宽屏电脑的高适配模拟外壳。在宽屏下呈现优雅的双列排布，在窄屏中自动聚合，极易作为独立 App 或 H5 端被集成或分享。
* **🛡 纯本地无痕安全**：不含外部动态广告或复杂的跨域存储，所有进度、星级、练习参数全自动保存在浏览器本地，提供高安全度的隐私呵护。

## 🚀 项目安装与部署

作为标准的 **Vite + React + TypeScript** 项目，它的启动非常简单：

1. **获取代码并进入根目录**：
   ```bash
   cd kidleap
   ```
2. **一键快速安装核心依赖**：
   ```bash
   npm install
   ```
3. **在本地启动极速开发服务**：
   ```bash
   npm run dev
   ```
   *服务将自动在 `http://localhost:3000` 渲染运行。*
4. **进行生产环境轻量化打包**：
   ```bash
   npm run build
   ```
   *打包完成后，纯静态资源会完美聚合至 `/dist` 目录中。*

## 💡 教育意义与陪伴建议

1. **拼音学习**：建议家长每天陪孩子使用“换一组”功能，跟读 15 分钟，随后完成一次听音练习，让孩子积累对拼音声母和韵母的直接语感。
2. **口算大冒险**：刚开始可以设置为 10 以内加减法、10 道题；当孩子自信满满时，再循序渐进在齿轮设置菜单内调大至 20 甚至 50 以内。
3. **古诗熟读**：本系统的“达标总星数”专门设置了 `10星 / 20星 / 30星 / 50星` 的里程碑，达到目标时，可以带孩子去户外活动作为一种小小的成果奖励，激发内驱力！
